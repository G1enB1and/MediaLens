## MediaLens v1.0.23

### Summary
This release completes the MediaLens rebrand across the shipped app and installer. It also updates the built-in updater and release links so future installs and version checks continue working from the renamed GitHub repository.

### Highlights
- Added the new MediaLens icon pack so the installer, app window, taskbar, and Start menu use the new branding.
- Updated the app title, header branding, and packaged output names so releases present a consistent MediaLens identity.
- Pointed version checks, installer downloads, and project links at the renamed G1enB1and/MediaLens repository.

### Notes
- The header logo now uses dedicated light and dark MediaLens variants while keeping the theme-aware sidebar toggle icons intact.
