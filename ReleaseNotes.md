## MediaLens v1.1.27

### Summary

This release makes local AI setup easier to understand and manage. MediaLens now gives clearer model install status, simpler setup wording, and direct install or uninstall controls without exposing backend details users do not need.

### Highlights

- Open Local AI Models and Status from the View menu whenever you want to check model readiness.
- See each model's purpose, size, install status, and available install or uninstall action in one cleaner window.
- Get more reliable first-run setup because MediaLens now verifies that the needed model files are actually present before showing a model as installed.

### Notes

- Local AI model files and dependencies are still installed only when selected. They are not bundled into the main installer.
