# What's New in MediaManagerX

## v1.0.1-alpha (Recent)

- **Masonry Layout Stability**: Fixed layout shifts by reserving space for media using aspect ratio placeholders.
- **Auto-Updater**: Integrated backend logic to check for and download updates from GitHub.
- **Scroll Reset**: Fixed issue where scroll position was not resetting when navigating pages.
- **GIF Performance**: Removed real-time GIF pausing to improve Lightbox responsiveness.
- **UI Refinements**: Enhanced glassmorphism effects and tinted native scrollbars.
- **Metadata Handling**: Improved EXIF/XMP harvesting and embedding for Windows compatibility.
